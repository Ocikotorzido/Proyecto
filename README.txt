comandos para empezar al abrir el proyecto

admin  --> para llegar loguearse en la apgina principal recomendacion usar usuario admin y contrasena admin

personas/menu  --> para acceder al menu sin loguearse o registrarse
accounts/register  --> para registrar en este caso de debe acceder a admin paraa autorizar registro
accounts/login -->para acceder y loguear este redirecciona al menu
accounts/logout --> para finalizar sesion
personas/listar_productos -->para visualizar la lista de productos
personas/agregar_producto  -->para registrar un producto
personas/buscar_producto -->para visualizar el producto por su codigo
personas/actualizar_producto --> para modificar algun producto
personas/eliminar_producto -->si queremos sacar algun producto de la lista
